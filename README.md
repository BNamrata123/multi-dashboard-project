#Multi Dataset Dashboard

## Live Demo
https://multi-dashboard-project.vercel.app/jobs

---

## Overview
This is a full-stack web application that visualizes multiple datasets using interactive charts.  
The dashboard provides insights into **Jobs, Entertainment, and Mental Health data** with dynamic updates and user-friendly UI.

---

## Tech Stack
**Frontend:**
- React.js
- Recharts
- Axios
- Bootstrap

**Backend:**
- Node.js
- Express.js

**Data Handling:**
- CSV Parser
- Multer (File Upload)

---

## Features
- Multiple dashboards (Jobs, Entertainment, Mental Health)
- Interactive charts (Bar, Pie, Line)
- Upload new datasets dynamically
- Real-time data updates (Auto refresh)
- Responsive and clean UI design
